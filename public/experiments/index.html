ASCII Demo Lab

A small lab of ASCII experiments: local motion, a skill-backed render, a direct MCP export, and two color-preserving ASCII-Motion imports built from an eye logo layered against animated wireframe meshes. The Codex host for this thread still did not expose the ascii-motion namespace as a first-class tool, but the server itself is present and working through raw JSON-RPC.
