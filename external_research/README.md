# External Research Intake

Put all vendor-generated deep-research outputs in `raw/`.

Recommended naming pattern:
- `chatgpt_deepresearch_report_01.md`
- `chatgpt_pro_reasoning_report_01.md`
- `gemini_deepresearch_report_01.md`
- `perplexity_research_report_01.md`
- `perplexity_claude_report_01.md`

If a vendor produced multiple files, keep them grouped by prefix.
Do not rename inside the content; only rename the file if needed.

Also add one row per file to `manifest.csv`.
